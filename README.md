# AI Resume Analyzer Agent

An AI-powered resume analyzer that evaluates resumes and provides feedback using Large Language Models (LLMs).

## Features

- Analyze resume content
- Identify strengths and weaknesses
- Provide improvement suggestions
- AI-powered evaluation

## Technologies Used

- Python
- OpenAI API
- AI / LLM

## Installation

1. Clone the repository
2. Install dependencies
3.  Run the project
